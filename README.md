# ECC83-daughterboard

A compact KiCAD module that allows low voltage designs to utilize ECC83/12AX7 valves.

## Contents

- `ECC83-daughterboard/` — KiCAD project files (schematic, PCB layout)
- `ECC83-daughterboard/w_vacuum.pretty/` — 9-pin noval socket footprints (GZC9-B)
- `ECC83-daughterboard/3dmodels/` — 3D STEP model for the S9CPC/VTB9-PT socket and generator script

## Socket

The board uses a **Belton VTB9-PT (S9CPC)** PCB-mount 9-pin noval socket, compatible with the ECC83/12AX7 B9A pinout. The custom STEP model can be regenerated by running `3dmodels/make_s9cpc_step.py` (requires [CadQuery](https://cadquery.readthedocs.io/)).
